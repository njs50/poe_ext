    Path of Exile Inventory Management and Crafting Helper (Chrome Plugin)

    Project maintained by njs50 Hosted on GitHub Pages — Theme by mattgraham

    What does it do?

    An extension to the chrome web browser that allows you to find crafting recipes and helps to organise your inventory.

    Installing the Extension

    1. Install from chrome web store.

    Using the Extension

    1. Log in to pathofexile.com with "Remember Me" checked.
    2. Click the "home" button on pathofexile.com then select "Stash Management"
    3. Follow the instructions that haven't been written yet!

    Updating the Extension

    (if you can't wait for chrome to autoupdate, which can take a couple of hours)

    1. In chrome type chrome://extensions into the address bar.
    2. Check 'Developer mode' in the top right corner
    3. Click 'Update extensions now'
